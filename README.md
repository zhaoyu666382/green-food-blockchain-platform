# 基于区块链的绿色食品交易平台

## 项目简介
本项目构建基于区块链与人工智能的健康饮食溯源与推荐平台，解决农产品供应链信息不透明、推荐不精准等问题。

- **刘翼** - 硬件集成、系统测试

## 技术栈
- **前端**: Vue.js 3 + Element Plus
- **后端**: Python Flask + MySQL
- **区块链**: Hyperledger Fabric / Ethereum
- **AI**: PyTorch + TensorFlow
- **数据库**: MySQL + Redis
