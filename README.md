# 基于区块链的绿色食品交易平台

> 东华理工大学 2025年大学生创新训练项目

## 📋 项目简介

当前美菜网、叮咚买菜、拼多多多多买菜等平台在推动农产品线上交易的同时，无法保证信息透明化，用户对产品卫生问题存在很大的担忧。

本项目构建基于区块链与人工智能的健康饮食溯源与推荐平台，通过Hyperledger Fabric整合全流程数据，引入ZKP与IPFS保障数据真实与隐私合规，实现可信溯源。

## 🛠️ 技术栈

### 前端技术
- **框架**: Vue.js 3
- **UI库**: Element Plus
- **状态管理**: Vuex
- **路由**: Vue Router

### 后端技术
- **语言**: Python 3.8+
- **框架**: Flask/Django
- **数据库**: MySQL + Redis
- **API**: RESTful API

### 区块链技术
- **联盟链**: Hyperledger Fabric
- **智能合约**: Chaincode (Go)
- **存储**: IPFS
- **隐私保护**: 零知识证明(ZKP)

### AI/ML技术
- **深度学习**: PyTorch, TensorFlow
- **推荐系统**: GNN + Transformer
- **图像识别**: CNN
- **知识图谱**: Neo4j

## 🚀 快速开始

### 环境要求
- Node.js 16+
- Python 3.8+
- MySQL 8.0+
- Git

### 安装步骤
```bash
# 1. 克隆项目
git clone https://github.com/你的用户名/green-food-blockchain-platform.git
cd green-food-blockchain-platform

# 2. 后端环境设置
cd backend
pip install -r requirements.txt
python run.py

# 3. 前端环境设置
cd ../frontend
npm install
npm run serve

# 4. 访问应用
# 前端: http://localhost:8080
# 后端API: http://localhost:5000
```
